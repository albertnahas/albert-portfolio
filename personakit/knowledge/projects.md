# Projects

## PersonaKit
**AI Agent Framework for Developers**

PersonaKit is Albert's developer toolkit for building AI chat agents with personality, knowledge, and memory.

**What it does:**
- Create AI agents with custom personalities and instructions
- Built-in RAG (Retrieval-Augmented Generation) from markdown files
- Memory persistence across conversations
- React hook (`useChatAgent`) for easy frontend integration
- Next.js starter template for instant deployment

**Tech Stack:** TypeScript, OpenAI, Vercel AI SDK, React, Next.js, RAG, Vector embeddings

**Links:**
- GitHub: https://github.com/albertnahas/persona-kit
- NPM: `npm install personakit`

**Why it exists:** Most AI agent frameworks are too complex or too opinionated. PersonaKit gives you the right abstractions without getting in your way.

## Aissist
**Personal AI Productivity Assistant**

Aissist is a personal assistant that helps you track tasks, goals, daily reflections, and provides AI-powered insights.

**What it does:**
- Task and goal tracking with AI-powered suggestions
- Daily journaling with semantic search across entries
- Smart reminders and contextual notifications
- Personal knowledge base built from your own notes

**Tech Stack:** React Native, Firebase, OpenAI, TypeScript

**Links:**
- GitHub: https://github.com/albertnahas/aissist
- Available on iOS and Android

**Why it exists:** Existing productivity tools don't adapt to you. Aissist learns from your habits and provides personalized guidance.

## React Agent Renderer
**UI Renderer for AI Agents**

A React component library for rendering AI agent responses with rich UI components.

**What it does:**
- Parse and render structured AI outputs (buttons, cards, forms)
- Support for custom component mapping
- Built-in components for common AI use cases
- TypeScript-first with full type safety

**Tech Stack:** React, TypeScript, Component libraries

**Links:**
- GitHub: https://github.com/albertnahas/react-agent-renderer
- NPM: `npm install react-agent-renderer`

**Why it exists:** AI agents often need to render more than just text. This library makes it easy to build rich, interactive AI interfaces.
